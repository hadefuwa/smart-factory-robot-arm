More Info: PLC Cube Color Bits
When a cube is detected, the system clears DB123.DBX32.0 to DB123.DBX32.3, then sets one bit for the detected cube:
`yellow_cube_detected` -> DB123.DBX32.0
`white_cube_detected` -> DB123.DBX32.1
`steel_cube_detected` -> DB123.DBX32.2
`alluminium_cube_detected` -> DB123.DBX32.3
When the PLC Completed Command is received, DB123.DBX32.0 to DB123.DBX32.3 are cleared again, ready for the next cube.
