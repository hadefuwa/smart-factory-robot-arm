More Info: PLC Cube Color Bits
When a cube is detected, the system clears DB124.DBX6.0 to DB124.DBX6.3, then sets one bit for the detected cube:
yellow_cube_detected -> DB124.DBX6.0
white_cube_detected -> DB124.DBX6.1
steel_cube_detected -> DB124.DBX6.2
alluminium_cube_detected -> DB124.DBX6.3
When the PLC Completed Command is received, DB124.DBX6.0 to DB124.DBX6.3 are cleared again, ready for the next cube.
